# Flue Example

This project demonstrates the usage of `flue` library.

## Getting Started

1.  Install dependencies in the root directory:
    ```bash
    pnpm install
    ```

2.  Run the development server:
    ```bash
    pnpm dev:example
    ```

## Features Demonstrated

*   **Container**: Styling with decoration (borderRadius, boxShadow).
*   **Flex Layouts**: Row, Column, Center.
*   **Text**: Flutter-like text styling.
*   **SizedBox**: Spacing elements.

## Project Structure

*   `src/App.vue`: Main demo application code.
*   `src/main.ts`: Application entry point.
